# Lagos-Air-Quality-Classifier-and-PM25-Predictor
This code makes use of the RandomForest and XGBoost classifiers and regressors to classify air quality into 4 different qualities and predict the amount of particulate matter in the atmosphere with a diameter of 2.5 micrometers (often seen in the data as pm25)
